jetSlider

jQuery plugin

Scroll down
or
use arrow keys

Features

  • Declarative init
  • Ability to change settings after init
  • Fullpage or inline sliders (with many instances on page)
  • Animating through the content between slides
  • Works on touchscreen mobile devices

More features

  • Slide element with size different than size of window won't break the slider
  • Slide elements can be positioned vertically and horizontally (or even diagonally)
  • CSS3 easing functions
  • IE8 support with fallback to JS animation
